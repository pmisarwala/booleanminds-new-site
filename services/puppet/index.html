Puppet Essentials

In this course, you will learn Puppet best practices and use Puppet to write classes and update and extend existing modules. Fundamentals is taught through a combination of lecture and relevant, real world, hands-on exercises. After completing this course, students will be able to deploy basic system configurations using best practices for Puppet in a Master-Agent setup.
