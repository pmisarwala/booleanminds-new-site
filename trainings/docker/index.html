Containerization with Docker

Learn how to use Docker, from beginner basics to advanced techniques. This course will help you gain insight into various aspects of containerization with Docker. You will learn how to use docker to deploy your application in a docker container. The course also introduces various container cluster management, and orchestration tools like Docker Compose and Docker Swarm.
